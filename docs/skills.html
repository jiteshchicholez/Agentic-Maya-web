Skill Definitions

Reusable agent capabilities with governance levels, input/output contracts, and audit support.

File Summarizer

MEDIUM

Extracts document key terms, structure, and intent to feed downstream validation.

Input Type Required Purpose
document_path string Yes Source file location
max_terms integer No Extraction limit
confidence_threshold float No Minimum result confidence
Output Type Description
key_terms array Extracted terms with scores
document_scope string Document summary
structured_fields object Parsed sections

Data Validator

LOW

Validates structured records against schema and compliance rule sets.

Input Type Required Purpose
data_input object/array Yes Records to validate
schema object Yes Validation rules
frameworks array Yes Compliance frameworks
Output Type Description
validation_result string PASS / FAIL / REVIEW_REQUIRED
violations_found array Detected issues
risk_score integer Compliance score

Report Writer

HIGH

Creates formal compliance reports with remediation steps and risk ratings.

Input Type Required Purpose
summary object Yes Summarized findings
violations array Yes Detected issues list
frameworks array Yes Report frameworks
Output Type Description
compliance_report object Formatted findings
remediation_steps array Ordered resolution tasks
risk_assessment object Overall risk summary

Semantic Search

MEDIUM

Performs vector search over knowledge sources to enrich results and identify related context.

Input Type Required Purpose
query string Yes Search prompt
knowledge_base array Yes Search corpus
top_k integer No Max hits
threshold float No Min relevance
Output Type Description
results array Ranked matches
relevance_scores array Confidence values
context_summary string Aggregated summary
